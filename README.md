# Mini Brick Studio PWA

Мобильное PWA-приложение для создания заказа на персональную рамку с мини-фигурками.

## Как загрузить на GitHub Pages

1. Создай новый репозиторий, например `minibrick-studio`.
2. Загрузи все файлы из этой папки в корень репозитория.
3. Открой `Settings` → `Pages`.
4. В `Build and deployment` выбери `Deploy from a branch`.
5. Branch: `main`, folder: `/root`.
6. Через минуту GitHub даст ссылку на приложение.

## Важно при обновлениях

После любых правок в файлах приложения увеличивай версию `CACHE_NAME` в `sw.js`, например:

```js
const CACHE_NAME = 'minibrick-studio-v2';
```

Иначе телефон может продолжать показывать старую версию из кэша.
