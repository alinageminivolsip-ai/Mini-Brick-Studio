# MiniBrick Studio 3D PWA

Простое PWA-приложение для GitHub Pages.

## Что внутри
- стартовый экран с логотипом
- 3D мини-фигурка на Three.js
- смена цвета кожи стрелками
- категории: кожа, причёски, торс, ноги, аксессуары, лицо
- создание до 20 человечков
- режим 3D фоторамки
- форма заказа с копированием

## Публикация
1. Распакуйте архив.
2. Загрузите все файлы в репозиторий GitHub.
3. Settings → Pages → Deploy from a branch → main → /root.
4. Save.

Важно: после любых правок увеличивайте CACHE_NAME в sw.js, например minibrick-3d-v2.
