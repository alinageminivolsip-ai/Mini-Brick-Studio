# MiniBrick Studio PNG MVP

Чистая версия приложения без Three.js и GLB.

## Что внутри
- index.html
- manifest.json
- sw.js
- logo.png
- icon-192.png
- icon-512.png
- assets/body/base-white.png

## Как обновить GitHub
1. В репозитории удали старые файлы, особенно папки models и старый index.html.
2. Загрузи содержимое этого архива в корень репозитория.
3. GitHub Pages: main / root.
4. После каждой правки увеличивай CACHE_NAME в sw.js.

## Проверка
Открой сайт с параметром версии, например:
https://alinageminivolsip-ai.github.io/Mini-Brick-Studio/?v=png1
